## BUGS ENCOUNTERED AND RESOLVED THIS SESSION

### 1. Duplicate `<script>` tag on Portfolio v5
- **Symptom**: `Uncaught SyntaxError: Unexpected token '<'`
- **Root cause**: HTML shell ended with `<script>` and assembler added another `<script>`. Browser saw nested script tags.
- **Fix**: str_replace to merge into single tag.
- **This bug has appeared twice.** If future rebuilds produce this error, check for duplicate `<script>` tags.

### 2. UTF-16LE encoding from PowerShell `>` redirect
- **Symptom**: grep silently returns no results on a file containing the search term.
- **Fix**: `iconv -f UTF-16LE -t UTF-8` or use `Out-File -Encoding utf8`.

### 3. Forecast tab lost during Omega rebuilds
- **Recovery**: Extracted from `git show eef406a:omega/index.html`, converted UTF-16→UTF-8, spliced 3 pieces (tab button, HTML view, CSS+module script).
- **Prevention**: Any rebuild must preserve the forecast `<script type="module">` block + CSS + HTML view + tab button.

### 4. Alpha Fee Monitor `.map()` crash
- **Symptom**: `Cannot read properties of undefined (reading 'map')` on XLS upload
- **Root cause**: Alpha's FD had wrong schema (`{by_type, by_ticker, raw}`) vs code expecting `{fee_records, trade_records, ticker_fees}`.
- **Fix**: Replaced FD schema, then rebuilt entire FD from actual broker XLS files.

### 5. Alpha Fee Monitor showing Omega data
- **Symptom**: Net Locate $3,641 and Commission $446 (Omega numbers) instead of $22,182 and $2,643 (Alpha numbers).
- **Root cause**: Initially embedded Omega's FD into Alpha since "same broker account". But date ranges differ (Omega Jul 2025+, Alpha Mar 2025+) and fee assumptions differ (20% vs 30%).
- **Fix**: Built Alpha's own FD by parsing the actual broker XLS files with Python (libreoffice CSV conversion + custom parsing).

### 6. Annotation editor event bubbling
- **Symptom**: Clicking "Add item" or "Save" buttons inside the withdrawal annotation editor caused the parent row to toggle closed.
- **Root cause**: Click events bubbled from buttons through to parent `onclick="feeToggleAnnotEditor(...)"`.
- **Fix**: Added `event.stopPropagation()` to all interactive elements inside the editor (10 instances: buttons, inputs, remove ×, container divs).

### 7. PDF too small
- **Symptom**: Downloaded PDF required 200% zoom to read.
- **Fix**: Changed from A4 landscape (297×210mm, scale 1.5) to A3 landscape (420×297mm, scale 3).

### 8. Broker XLS parsing — xlrd failure
- **Symptom**: `Expected "little-endian" marker` error when reading broker .xls files with xlrd.
- **Root cause**: Broker XLS files are Composite Document Format with non-standard markers.
- **Fix**: Used `libreoffice --headless --convert-to csv` to convert, then parsed CSV with Python.
